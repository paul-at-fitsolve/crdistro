This feature provides the Bootstrap grid system in the UI, to allow us to select column widths of content panes and arrange push/pull to change content ordering between mobile devices and desktops if neccessary. 

Further information on the wiki:
http://wiki.comicrelief.com/wiki/CR_Layout_V2#Panels_Style_Plugin

The Bootstrap grid system (and the only parts we are using) can be found at the below links:
http://getbootstrap.com/css/#grid-options
http://getbootstrap.com/css/#grid-column-ordering
http://getbootstrap.com/css/#grid-offsetting

Implementation Notes:
--------
In Panel Content using the UI you can select the column width of a pane in the grid along all screen widths in a mobile first fashion. XS, SM, MD, and LG. There is also a push/pull section allowing where the aforementioned screen widths can be used in the same way.

This section also makes use of the Match Height plugin, this allows content to be of the same height.
