CR Invalidate Varnish
=================================================

This module allows certain roles to request rate limited Varnish invalidations
Invalidation requests are made to a backend service, and successful requests are
stored.

Documentation for use can be found at
http://confluence.comicrelief.com/display/rnd15Documentation/Cache+and+Varnish+Configuration

Configuration page is at:
/admin/config/services/varnish_invalidation

Invalidation requests can be submitted at:
/admin/config/services/varnish_invalidation
